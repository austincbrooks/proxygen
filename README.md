# Proxygen

Proxygen (pronounced /PROX-eh-gen/) prepares media files for
video editing programs by optimizing source files and generating
proxies.

The code is licensed under GPLv3 and only uses FFmpeg and the
Python standard library (notably including `tkinter` for the GUI).
Releases include the source files for easy modification and
experimentation, but also include compiled executables for users
that don't want to install a Python runtime environment.
